1. 注册码云

2.安装Git工具

3. 目的协作开发一个项目
   3.1 在码云上创建一个仓库
		
   3.2 添加合作成员
   
   3.3 clone仓库代码到本地
		新建文件放在克隆的代码
		
		打开git bash -> git clone https://gitxxx
		
		按照提示输入用户名和密码
		
	3.4 拉取webpack空白模版，将空白模版的所有文件拷贝到本地clone下来的文件夹
   
    3.5 上传代码
		1）在仓库文件夹下打开git bash
		2）git add .              :将需要上传的代码添加到暂存区
		3）git commit -m 添加说明 :将暂存区代码提交到本地服务器
		4）git push   			   :将本地服务器代码提交到云服务器
    
	3.6 更新代码
		 git pull
		 
	3.7 提交或者更新过程可能会出现冲突
		git commit -m :提交本地服务器
		git pull	：拉取云服务器
		git status： 查看以下状态
		本地磁盘去修改有冲突的文件
		冲突文件修改好后则再次上传代码
